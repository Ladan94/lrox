FAQ

Häufig gestellte Fragen Frequently Asked Questions

Welche Sensoren werden verwendet?

Das System verwendet eine Kombination aus Kamera, GPS, IMU (Inertial Measurement Unit) und optional LIDAR für umfassende Umgebungserfassung. The system uses a combination of camera, GPS, IMU (Inertial Measurement Unit) and optional LIDAR for comprehensive environment perception.

Ist eine Kalibrierung erforderlich?

Das System kalibriert sich automatisch beim ersten Start. Bei Hardware-Änderungen ist eine Neukalibrierung empfohlen. The system calibrates automatically on first startup. Recalibration is recommended after hardware changes.

Wie werden meine Daten geschützt?

Alle Datenverarbeitung erfolgt on-device. Keine personenbezogenen Daten werden an Cloud-Server übertragen. All data processing happens on-device. No personal data is transmitted to cloud servers.

Ist der Smart Helmet mit allen Fahrradhelmen kompatibel?

Unser Smart Helmet ist ein komplettes System. Für bestehende Helme bieten wir ein Nachrüstkit an. Our Smart Helmet is a complete system. For existing helmets, we offer a retrofit kit.

Gibt es rechtliche Hinweise für die Straßenzulassung?

Das System ist als Assistenzsystem konzipiert und ersetzt nicht die Aufmerksamkeit des Fahrers. Es entspricht den relevanten EU-Richtlinien. The system is designed as an assistance system and does not replace rider attention. It complies with relevant EU directives.

Wie erfolgen Software-Updates?

Updates werden über eine mobile App bereitgestellt, die eine sichere Verbindung zum Gerät herstellt. Updates are provided through a mobile app that establishes a secure connection to the device.

Was passiert bei Systemausfall?

Bei kritischen Fehlern warnt das System den Nutzer und schaltet in einen sicheren Modus. Die Grundfunktionen des Fahrrads bleiben uneingeschränkt nutzbar. In case of critical errors, the system warns the user and switches to a safe mode. The basic functions of the bicycle remain fully usable.

Wie lange werden Daten gespeichert?

Fahrtdaten werden maximal 24 Stunden lokal gespeichert, es sei denn, der Nutzer exportiert sie explizit. Ride data is stored locally for a maximum of 24 hours, unless the user explicitly exports it.
