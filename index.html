FAQ

Frequently Asked Questions

Which sensors are used?

The system uses a combination of camera, GPS, IMU (Inertial Measurement Unit) and optional LIDAR for comprehensive environment perception.

Is calibration required?

The system calibrates automatically on first startup. Recalibration is recommended after hardware changes.

How is my data protected?

All data processing happens on-device. No personal data is transmitted to cloud servers.

Is the Smart Helmet compatible with all bicycle helmets?

Our Smart Helmet is a complete system. For existing helmets, we offer a retrofit kit.

Are there legal requirements for road approval?

The system is designed as an assistance system and does not replace rider attention. It complies with relevant EU directives.

How are software updates performed?

Updates are provided through a mobile app that establishes a secure connection to the device.

What happens in case of system failure?

In case of critical errors, the system warns the user and switches to a safe mode. The basic functions of the bicycle remain fully usable.

How long is data stored?

Ride data is stored locally for a maximum of 24 hours, unless the user explicitly exports it.
